# 🚩 원티드 프리온보딩 백엔드 인턴십 사전 과제

## 서비스 소개
- 본 서비스는 기업의 채용을 위한 웹 서비스입니다.
- 회사는 채용공고를 생성하고, 사용자는 원하는 채용에 지원할 수 있습니다.

<br>

## 사용 기술
- 언어 및 프레임워크 : **Java & SpringBoot**
- ORM : **Spring Data JPA**
- DataBase : **MySQL**

<br>
